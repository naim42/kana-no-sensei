Hi there!

I'm a French teenager who likes to study the japanese language on his free time.
I developed this tool to help me learn hiragana and katakana (the two main japanese alphabets).
It's very simple, just type the romaji transliteration for the kana(s) you see.
You can choose to show hiragana or katakana, diacritics and digraphs.
The source code for this project is available on GitHub under the MIT License.
